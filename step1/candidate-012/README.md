# One-Arm Slot Machine — Roast the AI

Lightweight vanilla web slot machine that gently mocks AI while you win and spend tokens.

How to run

- Open [index.html](index.html) in any modern browser (no build or server required).

Gameplay

- You start with 50 tokens. Each spin costs 5 tokens.
- Match rules: Triple = big win, pair = small win, special triple 🤖 = jackpot.
- Tokens persist in `localStorage`.

Files

- [index.html](index.html) — main UI
- [styles.css](styles.css) — styling
- [script.js](script.js) — game logic
