# One-Arm Slot Machine — Roast the AI

Simple vanilla web slot machine that playfully mocks AI predictions. Open `index.html` in your browser to run.

Controls:
- Pull Lever: costs 1 token per spin.
- Buy Prompt: spend 10 tokens for a snarky log message.
- Train AI: spend 20 tokens to "train" the AI (just flavor).

Start with 100 tokens. Wins vary: matching three symbols gives larger rewards.

To run locally, open the file `index.html` in a browser (double-click or use a local dev server).
