# One-Arm Slot Machine — Candidate 015

Simple vanilla HTML/CSS/JS slot machine that gently mocks AI.

How to run:

1. Open [index.html](index.html) in a browser.
2. Use the "Add 5 Tokens" button to add tokens, then press "Pull the arm!" to spin.

Features:
- Token economy persisted in `localStorage`.
- Fun AI taunts when you lose or when you "Ask AI".
