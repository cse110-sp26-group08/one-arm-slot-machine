## About This Experiment

This project explores how Large Language Models (LLMs) respond to iterative refinement prompts when generating web applications. We generated 50 slot machine websites using Codex, then systematically refined them through 5 stages of prompt iteration, selecting the best candidates at each stage based on a comprehensive rubric. The experiment demonstrates how sensitive LLMs are to prompt engineering and the importance of specifying elements to preserve during refinement.

## Model and Harness

- **Model:** Gpt-5.4 low reasoning
- **Harness:** VSCode Extension (Codex)
